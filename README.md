## Dependencies

- Xlibre or X11 (duh)
- feh
- flameshot (optional, only for screenshots)

### Fonts required

- ttf-jetbrains-mono-nerd
- ttf-nerd-fonts-symbols 

## Installation
```bash
git clone https://github.com/QuantaDude/dgbwm
cd dgbwm
chmod +x install_dgbwm.sh
sudo ./install_dgbwm.sh
```
